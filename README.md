# 歌唱譜メーカー

ボーカリストやボーカルトレーナーが、歌詞・母音・コード・歌唱記号をすばやく配置できるWebアプリです。

## 主な機能

- 歌唱譜キャンバスへのドラッグアンドドロップ配置
- ショートカットキーで歌唱記号を選択
- ビブラート、ブレス、しゃくり、フォール、アクセント、ロングトーン、強弱、コード、歌詞の配置
- 漢字まじり歌詞からひらがなへの変換
- 手動で読みを修正できる編集欄
- ひらがな歌詞から母音への変換
- コード記号の配置と編集
- ローカル保存、JSON書き出し、印刷

## 開発

```bash
npm install
npm run dev
```

http://localhost:3000 を開きます。

## ビルド

```bash
npm run build
```
